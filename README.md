# CSEC 2300 - Foundations of Cyber Security (public course materials)

University of the Incarnate Word - Dr. Gonzalo D Parra

Open **index.html** (the resource browser). It lists the labs and documents and renders them in place; slides, the course hub, the outline, the fillable capstone forms, and the handouts open in a new tab. Press **p** on any page to download or print it.

Generated by `build_public_site.py` from the course sources. Student-facing material only: answer keys, exam banks, autograders, instructor handout keys, and slide speaker notes are never published. The graded lab code (starters plus autograder) is delivered privately per student, not here.
