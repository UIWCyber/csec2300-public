# CSEC 2300 - Foundations of Cyber Security (public course materials)

University of the Incarnate Word - Dr. Gonzalo D Parra

Public, student-facing resources for CSEC 2300. Open **`index.html`** (the resource
browser) for a single page that lists the labs and every document and renders them in
place, or browse the folders directly.

## What is here

- `index.html` - resource browser: labs, lab documents, study guides, capstone documents,
  and links to the slides, course hub, and outline.
- `course-hub.html` - the weekly student hub (schedule, labs, grading, resources).
- `course-outline.html` - the printable course outline.
- `slides/` - the lecture decks (HTML, present in the browser full screen).
- `labs/<lab>/` - each lab's `README.md`, `HINTS.md`, and `STUDENT-GUIDE.md` (and starter
  notes). These describe what to do and how; the graded lab repositories are distributed
  separately and privately.
- `study-guides/` - midterm and final study guides (concepts and self-checks, no exam items).
- `capstone/` - proposal template, presentation rubric, readiness checklist, team-formation
  guide, and peer-feedback form.

## What is intentionally NOT here

Answer keys, quiz and exam question banks, autograders, and instructor-only handout keys are
kept in a separate private repository. Nothing in this public repository reveals graded
answers.

## Hosting

Served with GitHub Pages. If Pages is enabled on this repository, the site is live at the
Pages URL shown in the repository settings; `index.html` is the landing page.
