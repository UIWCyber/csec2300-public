Capstone Grading Rubric

CSEC 2300 Foundations of Cyber Security  |  Instructor: Dr. Gonzalo D Parra
One rubric for the whole capstone (20% of the course grade), one section per graded component: Proposal 3%, Project Implementation 5%, Final Presentation 5%, GitHub Repository and Project Documentation 4%, GitHub Wiki Final Report 3%. Within each section, criteria weights add to 100%. Pick a level for every criterion; the form converts each section score to its course points and totals everything automatically. Levels award 0%, 10%, 30%, 60%, 80%, or 100% of a criterion's weight (so a 10% criterion offers 0 / 1 / 3 / 6 / 8 / 10).

Individual grades (contribution-scaled)

Individual = Team x N x Contribution %, capped per component

Enter each student's contribution percent from the GitHub record (story points earned divided by the team's total, per the Individual Contribution Summary Table in the README, verified against the board and commit history). Each student's score is the team's component scores scaled by N members times their contribution, capped at each component's maximum: contributing your fair share earns the full team score, doing more cannot exceed the maximums, and contributing nothing earns zero.
